# discord_py
## Для пользователей
1. Скачать архив репозитория с данного сайта.
2. Создать аккаунт бота через сайт https://discord.com/developers.
3. Задать разрешения бота на сообщения сервера, доступ к серверу и информации участников.
4. Пригласить бота на сервер.
5. Установить нужные плагины.
6. Запустить бота, введя токен с сайта.

## Для создателей плагинов
1. Для создания плагина нужна отдельная директория, в котором будут файл программы (program.py) и конфиг (config.json).
2. В конфиге используется структура JSON, и в нем должны быть 4 набора значений (Name, Author, Version, CoreVersion). При том версия указанная в CoreVersion будет зависеть от версии программы бота.
3. Файл program.py должен состоять только из функций улавливание событий или обычных функций (как показано в плагине Example1). Файл программы не должен содержать bot.run(TOKEN).
